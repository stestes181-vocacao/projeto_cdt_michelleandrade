🐾 Patinhas do Bem
Bem-vindo ao Patinhas do Bem, um projeto desenvolvido para ajudar animais a encontrarem um novo lar!
A plataforma foi criada com foco em facilitar a adoção de pets de forma simples, organizada e acessível.
🌐 Acesse o projeto:
patinhasdobemm.vercel.app⁠�
📌 Sobre o Projeto
O Patinhas do Bem é um site de adoção de animais onde os usuários podem visualizar pets disponíveis, conhecer informações sobre eles e incentivar a adoção responsável.
O projeto foi desenvolvido utilizando tecnologias web e possui uma interface simples e intuitiva.
✨ Funcionalidades
🐶 Visualização de pets disponíveis para adoção
📷 Galeria com imagens dos animais
📄 Informações sobre cada pet
🎨 Interface amigável e organizada
🌐 Projeto hospedado online pela Vercel
